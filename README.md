## Installation

```bash
npm install
```

## Usage

```bash
npm start
```

Go to http://localhost:3000 to explore your series and movies.


## Requirements

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com/)
- [tmdb](https://www.themoviedb.org/)
